# 🥔 AI 土豆兄弟 (Brotato Lite) - 无尽变异首领战版

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的轻量级肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无任何第三方依赖，完美适配 PC 端（键盘）与移动端（手机虚拟摇杆）双端操作。

## 🎮 游戏核心玩法与波次矩阵
1. **走位生存**：红色敌人会从四周源源不断地涌向中央，玩家需要控制黄色土豆进行走位拉扯。
2. **自动锁敌**：角色会自动锁定并射击距离最近的敌人，无需手动瞄准。
3. **磁性收集**：消灭敌人后会掉落经验石（普通怪掉绿色[1点经验]；精英与首领爆金色[3点经验]），进入吸附半径后自动飞向玩家。
4. **🛍️ 动态去重商店**：升级时游戏暂停并弹出三选一商店。高稀有度道具“散弹枪改装”未解锁时激活散射，**已解锁则使弹道数无限 +1 叠加**，支持后期全屏开花式暴力割草。
5. **💀 阶梯式无尽变异矩阵 (新增)**：
   * **0 ~ 30 斩**：普通红色怪群热身。
   * **30 ~ 60 斩**：**亮蓝色精英怪**混入怪群。普通怪与精英怪解锁**红绿灯蓄力冲刺异能**（原地锁死蓄力后瞬间爆发数倍速度发起暴走突袭，造成双倍伤害）。**精英怪死亡后会瞬间分裂出 2 只高移速的脆皮橘色蜘蛛怪**。
   * **60 斩整点 / 120 斩 / 180 斩...**：小怪不停，**紫色终极恶魔首领 (Boss)** 缓缓降临！Boss 体型为普通怪的 4 倍，血量极厚（随 Loop 轮次翻倍），顶部同步激活**带呼吸灯闪烁警告的紫红渐变巨型血条**。
   * **🥊 斩首奖励与无尽套娃**：成功击杀 Boss 后，血条隐去，原地爆散大量金色高额经验石并**强制额外赠送一次免费商店机会**。游戏不结束，怪物属性指数级跃迁，直接进入下一轮更凶残的 Loop 维度循环挑战！

## 🚀 如何运行游戏？

### 💻 途径一：电脑端直接运行（最简单）
直接双击 `index.html` 文件，它会自动通过您电脑的默认浏览器打开，使用键盘 **`W, A, S, D`** 或 **方向键** 即可控制移动。

### 📲 途径二：手机端局域网畅玩（推荐测试触控）
如果您安装了 Python 环境，可以通过以下步骤将电脑变为局域网服务器，用手机连接游玩：

1. 进入 `brotato_game` 文件夹，在地址栏输入 `cmd` 并回车打开命令行窗口。
2. 在窗口中输入以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 保持该窗口不要关闭，重新打开一个全新的 `cmd` 窗口，输入 `ipconfig` 查询您电脑的 **IPv4 地址**。
4. 确保手机与电脑连接在**同一个 Wi-Fi** 下，在手机浏览器中输入网址即可开始游玩：
   ```text
   http://您的电脑IP:8000/index.html
   ```

## 🛠️ 技术亮点
* **🚦 红绿灯冲刺状态机技术 (新增)**：为怪物引入了 `walk -> charge -> dash` 三段式行为树算法。蓄力时通过 Canvas 实时渲染白光与红色警示圈，冲刺时锁定切线向量滑行，极大提升了动作肉鸽的博弈感。
* **🕷️ 内存裁剪与实体分裂算法 (新增)**：在子弹命中判定中植入死亡解体钩子。当特定唯一标识的 `elite` 实体被销毁时，利用循环动态向数组追加高速 `spider` 实体，实现防线的二次崩溃压力。
* **🛡️ 健壮性空值安全防护 (新增)**：在 `game.js` 的 DOM 交互中全面配置了防御性空值校验（Optional Chaining 变异逻辑），确保在跨波次、高密度刷新 UI 时绝对不发生页面卡死或句柄丢失。
* **零依赖（Zero Dependencies）**：纯原生前端技术栈，加载速度极快。
* **斜向速度归一化**：位移速度经过数学公式归一化处理，避免了斜向移动速度变快的 BUG。
* **多端自适应布局**：彻底禁用了移动端的双击缩放、手势滚动与长按菜单，带来纯正的独立 App 游玩体感。
