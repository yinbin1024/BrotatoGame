# 🥔 AI 土豆兄弟 (Brotato Lite) - 第二武器与速度平衡版

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的高硬核度动作肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无任何第三方依赖。

在本版本中，游戏迎来了具有里程碑意义的**第二武器里程碑解禁系统**，并针对高轮次无尽挑战中怪物的“移速失控”痛点进行了科学的数学衰减软上限微调，大幅提升了动作微操拉扯的策略性。

---

## 🎮 游戏核心机制与波次矩阵

### 1. 🚦 阶梯式无尽变异矩阵
* **0 ~ 30 斩**：普通红色怪群热身，熟悉双端（PC键盘/手机虚拟摇杆）位移手感。
* **30 ~ 50 斩**：混入**亮蓝色精英怪**。普通怪与精英怪共同解锁**红绿灯蓄力冲刺异能**（原地锁死蓄力并发出白光红圈警示，随后突发 2.2 倍爆发速度发起冲刺）。精英怪死亡后**原地解体并分裂出 2 只高移速脆皮蜘蛛怪**。
* **50 斩相对安全冷却区**：击杀完上一个 Boss 后，必须再次亲手斩杀 50 只怪物才会激活下一个 Boss，彻底杜绝多只 Boss 重叠刷脸的灾难体验。
* **👑 Boss 斩首狂欢**：每 50 斩降临**“第 X 轮 · 灾变恶魔宿主”**（血量随轮次阶肽暴涨），击杀后原地爆散 **20 颗金色大宝石**，触发 **3秒黄金无敌暴走**（伴随能量冲击波外扩，撞怪直接 **`💥 CRUSH!`** 秒杀）。

### 2. ⚔️ 斩首成就：自动解禁第二武器【环绕飞刀】
游戏打破了只能在商店抽武器的传统规则，引入了**阶段性里程碑解锁机制**：
* **解禁条件**：当玩家在战场上展现极致走位，成功**亲手斩杀第 3 代灾变恶魔宿主（累计 150 斩左右）**时，系统会瞬间在空中弹出紫色成就特效，并**永久、免费赠送第二武器：环绕飞刀**。
* **旋转机制**：解禁后，1 把紫色环绕飞刀会自动在体内凝聚，以土豆主角为中心，按 60 像素的防御半径像**微型行星带**一样进行 360 度高频旋转切割。
* **能量守恒衰减与力竭碎裂**：飞刀继承玩家实时的基础攻击力。为了防止站撸，飞刀击中怪时不会消失，但**每切割中一个怪，该飞刀自身携带的攻击力永久衰减 30%**。
* **碎裂与自动凝聚循环**：当某把飞刀的攻击力在怪堆里高频连切、**跌落至小于 1 时，该飞刀会瞬间发生 `✨ SHATTER!` 碎裂解体**。当场上飞刀不足时，主角需要走位拉扯 1.5 秒（90帧）等待飞刀在体内重新凝聚。
* **奖池解禁**：在成功放倒 3 代 Boss 之前，**升级商店里绝对不存在任何飞刀强化**；一旦飞刀解禁，普通商店将**动态注入全新的 `🗡️ 飞刀增幅卷轴` 词条**，玩家后续每次抽取，身边的旋转飞刀上限永久 **+1** 把它堆成巨型绞肉机！

### 3. ⚖️ 速度平衡性抑制算法（反粘人微调）
针对后期高轮次（如第 15 轮以上）怪物移速无限线性膨胀、导致不选移速词条的玩家毫无操作空间的痛点，本版本进行了底层物理调校：
* **平方根软上限曲线**：将怪物的无脑追击速度公式重构为 `Math.sqrt(currentLoop) * 0.12`。这让怪物速度在中后期无限趋于平缓。
* **冲刺爆发降级**：将怪物蓄力红光后的闪电冲刺爆发倍率从不合理的 3.5 倍下调至 **2.2 倍**，给极限制空走位留出了完美的变向闪避身位。

### 4. 🧪 每级自动触发：动态生命体感保底
游戏移除了原本占用商店格子的鸡肋血清，将其转化为每级自动触发的“伤势博弈”保底福利：
* **满血升级**：直接永久增加**【当前等级数】的生命上限**（例如 6 升 7 级满血，上限永久 +7）。
* **残血升级**：触发伤势紧急治疗，**立刻恢复当前失去血量的 20% 气血**（雪中送炭，极高提升低血容错）。

### 5. 💥 弹道能量守恒与熔断矩阵（硬核动态平衡）
* **暴击与穿透互斥**：单发子弹在单次碰撞中，暴击与穿透无法同时触发。
* **🛡️ 超上限穿透矩阵**：穿透率提升 35%（史诗红装独占）。穿透后子弹永久无法再次暴击，且子弹基础伤害直接砍半（衰减 50%）。当穿透率 \(\ge 100\%\) 时，整数部分直接转化为【必定穿透次数】。
* **🕷️ 恶魔裂变散射**：暴击时怪身解体爆散出碎弹（初始 1 分 2）。分裂子弹的总攻击力由上一发子弹平均分配，且状态锁死：永久无法再次穿透、永久无法再次暴击。
* **⚠️ 数值底线熔断**：任意子弹在经历层层衰减后，一旦基础攻击力 \(< 1\)，该子弹在触发碰撞的瞬间将强制被剥夺所有的暴击与穿透特权，击中当前怪物后直接原地销毁。
