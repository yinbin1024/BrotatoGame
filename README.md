# 🥔 AI 土豆兄弟 (Brotato Lite) - 商业级核心平衡版

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的高硬核度动作肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无任何第三方依赖。

在本版本中，游戏引入了**动态伤势保底、双奖池彻底隔离、高精度弹道能量守恒衰减、超上限穿透矩阵、以及整点升级三选二**等极具独立游戏质感的底层算法，将割草的爽感与局内属性配比的博弈感拉到了全新高度。

## 🎮 游戏核心机制与波次矩阵

### 1. 🚦 阶梯式无尽变异矩阵
* **0 ~ 30 斩**：普通红色怪群热身，熟悉双端（PC键盘/手机虚拟摇杆）位移手感。
* **30 ~ 50 斩**：混入**亮蓝色精英怪**。普通怪与精英怪共同解锁**红绿灯蓄力冲刺异能**（原地锁死蓄力并发出白光红圈警示，0.5秒后爆发 3.5 倍滑行速度发起双倍伤害突袭）。精英怪死亡后**原地解体并分裂出 2 只高移速的脆皮橘色蜘蛛怪**。
* **50 斩相对安全冷却区**：击杀完上一个 Boss 后，必须再次亲手斩杀 50 只怪物才会激活下一个 Boss，彻底杜绝多只 Boss 重叠刷脸的灾难体验。
* **👑 Boss 斩首狂欢**：每 50 斩降临**“第 X 轮 · 灾变恶魔宿主”**（体型大4倍，血量随轮次阶梯暴涨），激活专属渐变大血条。击杀后原地爆散 **20 颗高额金色大宝石**，并触发 **3秒钟“黄金无敌暴走状态”**（土豆全身化为纯金并伴随金色能量冲击波向外猛烈扩散，碰到的任何怪都会在一连串 **`💥 CRUSH!`** 特效中被**直接秒杀**）。

### 2. 🧪 每级自动触发：动态生命体感保底
游戏移除了鸡肋的被动防具商店，将其转化为每级自动触发的“伤势博弈”保底福利：
* **满血升级**：直接永久增加**【当前等级数】的生命上限**（例如 6 升 7 级满血，上限永久 +7，越到后期雪球变坦越强）。
* **残血升级**：触发伤势紧急治疗，**立刻恢复当前失去血量的 20% 气血**（雪中送炭，极高提升前中期容错率）。

### 3. 🛍️ 规则解耦与整点升级“三选二”
* **彻底剥离隔离池**：普通道具与红色史诗神装在代码底层完全独立。平时升级商店**史诗装备绝不准乱入**，确保了常规发育期的纯净度。
* **🎁 5 倍整点特惠**：每当玩家升到 **5、10、15...等 5 的倍数等级**时，商店购买额度破格升级为 **“三选二”**，允许在一张货架上同时打包带走两件普通强化，迎来属性爆发期。
* **👑 首领商店**：击杀 Boss 后呼出的特殊商店**维持严苛的“三选一”终极抉择**，但货架第一个格子**锁定刷出可无限重复抽取的红色史诗机制神装**。

### 4. 💥 弹道能量守恒与熔断矩阵（硬核动态平衡）
为了防止中后期无限穿透与无限裂变导致的数据通胀卡死，游戏植入了极其严密的弹道约束算式：
* **暴击与穿透互斥**：单发子弹在单次碰撞中，**暴击与穿透无法同时触发**。
* **🛡️ 超上限穿透矩阵（可無限疊加）**：每次抽取红色穿透机制使穿透率提升 35%。
  * 穿透率 \(< 100\%\) 时，子弹概率穿透，穿透后**永久无法再次暴击，且子弹基础伤害直接砍半（衰减 50%）**。
  * 穿透率 \(\ge 100\%\) 时，**整数部分直接转化为【必定穿透次数】**，余数部分作为下一个怪的概率判定（例如 \(135\%\) 穿透率 = 必定穿透 1 个怪，撞第 2 个怪时有 35% 几率继续穿透）。
* **🕷️ 恶魔裂变散射（可無限疊加）**：初始暴击时怪身解体爆散出 **2 发** 碎弹（后续每抽一次红装分裂数 \(+1\)）。**分裂出的子弹总攻击力由上一发子弹平均分配，且状态锁死：永久无法再次穿透、永久无法再次暴击**，彻底锁死死循环。
* **⚠️ 数值底线熔断**：**任意子弹在经历层层衰减后，一旦基础攻击力 \(< 1\)，该子弹在触发碰撞的瞬间将强制被剥夺所有的暴击与穿透特权，击中当前怪物后直接原地销毁。** 这一机制逼迫玩家必须在商店里平衡“攻击力数值”与“弹道数量”，具有极高的策略深度。

## 🚀 如何运行游戏？

### 💻 途径一：电脑端直接运行
直接双击 `index.html` 文件即可用默认浏览器秒开，使用键盘 **`W, A, S, D`** 或 **方向键** 即可控制移动。

### 📲 途径二：手机端局域网测试（完美体验原生摇杆）
1. 进入 `brotato_game` 文件夹，在地址栏输入 `cmd` 并回车打开命令行。
2. 输入命令启动 Python 局域网广播服务器：
   ```bash
   python -m http.server 8000
   ```
3. 保持窗口不关，新建一个 cmd 输入 `ipconfig` 查出你电脑的 **IPv4 地址**（如 `192.168.1.5`）。
4. 手机连上**同一个 Wi-Fi**，在手机浏览器里输入网址即可畅玩：
   ```text
   http://您的电脑IP:8000/index.html
   ```
