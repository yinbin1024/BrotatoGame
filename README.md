# 🥔 AI 土豆兄弟 (Brotato Lite) - 暴击飘字与首领狂欢版

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的轻量级肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无任何第三方依赖，完美适配 PC 端（键盘）与移动端（手机虚拟摇杆）双端操作。

## 🎮 游戏核心玩法与波次矩阵
1. **走位生存**：红色敌人会从四周源源不断地涌向中央，玩家需要控制黄色土豆进行走位拉扯。
2. **自动锁敌**：角色会自动锁定并射击距离最近的敌人，无需手动瞄准。
3. **磁性收集**：消灭敌人后会掉落经验石（普通怪掉绿色[1点]；精英与首领爆金色[3点]），进入吸附半径后自动飞向玩家。
4. **🛍️ 动态去重商店**：升级时游戏暂停并弹出三选一商店。可提升攻击、移速、攻速、暴击率。高稀有度道具“散弹枪改装”未解锁时激活散射，**已解锁则使弹道数无限 +1 叠加**。
5. **🎯 暴击与吸血续航（新）**：
   * **暴击系统**：购买“鹰眼准星”可堆叠暴击率。触发时伤害翻倍，并在怪物头顶跳出**带有金色加粗描边、鲜红色高亮的 `💥 伤害!` 震撼视觉飘字**。
   * **克制吸血**：普通奖池不再提供吸血。击杀 Boss 选取红色史诗神装可每次微幅提升 3% 吸血率（上限锁死 25%）。引入 **0.5秒内置冷却时间(ICD)** 防站撸，触发时头顶会悠闲飘起**翠绿色的 `❤️ +1` 治愈反馈字**。
6. **💀 阶梯式无尽变异矩阵**：
   * **30 ~ 50 斩**：混入**亮蓝色精英怪**，解锁**红绿灯蓄力冲刺异能**（原地锁死蓄力红光警示，随后瞬间爆发双倍伤害突袭）。精英怪死亡后**分裂出 2 只高移速小蜘蛛怪**。
   * **50 斩相对安全冷却区**：击杀完上一个 Boss 后，必须再次亲手斩杀 50 只怪物才会激活下一个 Boss，彻底杜绝多只 Boss 重叠刷脸的灾难体验。
   * **Boss 斩首狂欢（新）**：每 50 斩降临**“第 X 轮 · 灾变恶魔宿主”**，顶部同步激活专属巨型血条。击杀后原地爆散 **20 颗高额金色大宝石**，触发 **3秒钟“黄金无敌暴走状态”**（土豆全身化为纯金并伴随**一圈圈动态金色能量冲击波向外猛烈扩散**，手枪射速开挂）。随后弹出**锁定刷出红色史诗可无限叠加神装**的专属商店。

## 🚀 如何运行游戏？

### 💻 途径一：电脑端直接运行（最简单）
直接双击 `index.html` 文件，它会自动通过您电脑的默认浏览器打开，使用键盘 **`W, A, S, D`** 或 **方向键** 即可控制移动。

### 📲 途径二：手机端局域网畅玩（推荐测试触控）
如果您安装了 Python 环境，可以通过以下步骤将电脑变为局域网服务器，用手机连接游玩：

1. 进入 `brotato_game` 文件夹，在地址栏输入 `cmd` 并回车打开命令行窗口。
2. 在窗口中输入以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 保持该窗口不要关闭，重新打开一个全新的 `cmd` 窗口，输入 `ipconfig` 查询您电脑的 **IPv4 地址**。
4. 确保手机与电脑连接在**同一个 Wi-Fi** 下，在手机浏览器中输入网址即可开始游玩：
   ```text
   http://您的电脑IP:8000/index.html
   ```

## 🛠️ 技术亮点
* **📊 Canvas 高性能双重特效渲染 (新增)**：利用 Canvas 原生位图混合技术，通过 `invincibleTimer % 30` 运算实时解算抛物线扩散路径，实现了不依赖 CPU 额外计算的“无敌超载冲击波”与“伤害/治愈彩色实时飘字”粒子生命周期销毁机制。
* **⚖️ ICD (内置冷却时间) 与硬上限熔断 (新增)**：针对数值大通胀引入了类似商业端游的 ICD 频率限制器。通过 `currentFrame - lastLeechTime > 30` 在底层对高频多发散弹枪的吸血吞吐量进行强制熔断，配合 `Math.min(0.25)` 限制，确保了游戏长线的博弈快感。
* **🔒 奖池剥离与权重干预机制 (新增)**：重构了商店生成逻辑，利用数组 `filter` 算法将普通道具和史诗道具彻底隔离，根治了神装在普通升级时提前乱入的 Bug，并在 Boss 战后提供精准的概率加权展示。
* **🚦 红绿灯冲刺状态机技术**：为怪物引入了 `walk -> charge -> dash` 三段式行为树算法，极大提升了动作肉鸽的微操感。
* **零依赖（Zero Dependencies）**：纯原生前端技术栈，加载速度极快。
* **斜向速度归一化**：位移速度经过数学公式归一化处理，避免了斜向移动速度变快的 BUG。
