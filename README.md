# 🥔 AI 土豆兄弟 (Brotato Lite) - 武器与商店迭代版

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的轻量级肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无 any 第三方依赖，完美适配 PC 端（键盘）与移动端（手机虚拟摇杆）双端操作。

## 🎮 游戏核心玩法
1. **走位生存**：红色敌人会从四周源源不断地涌向中央，玩家需要控制黄色土豆进行走位躲避。
2. **自动锁敌**：角色会自动锁定并射击距离最近的敌人，无需手动瞄准。
3. **磁性收集**：消灭敌人后会掉落绿色经验石，当玩家靠近时，经验石会自动产生磁性并向玩家飞行。
4. **🛍️ 商店强化**：收集足够的经验石即可升级。升级时游戏会自动暂停并弹出三选一商店。
5. **💥 散弹枪矩阵（新）**：在商店中有几率刷出高稀有度道具“散弹枪改装”。购买后，武器将由单发进化为**前方扇形区域 3 发子弹同时喷射**。为了保证肉鸽玩法的纯正体验，该武器**一经购买便会从道具池中动态下架**，绝不占用后续升级位。

## 🚀 如何运行游戏？

### 💻 途径一：电脑端直接运行（最简单）
直接双击 `index.html` 文件，它会自动通过您电脑的默认浏览器打开，使用键盘 **`W, A, S, D`** 或 **方向键** 即可控制移动。

### 📲 途径二：手机端局域网畅玩（推荐测试触控）
如果您安装了 Python 环境，可以通过以下步骤将电脑变为局域网服务器，用手机连接游玩：

1. 进入 `brotato_game` 文件夹，在地址栏输入 `cmd` 并回车打开命令行窗口。
2. 在窗口中输入以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 保持该窗口不要关闭，重新打开一个全新的 `cmd` 窗口，输入 `ipconfig` 查询您电脑的 **IPv4 地址**。
4. 确保手机与电脑连接在**同一个 Wi-Fi** 下，在手机浏览器中输入网址即可开始游玩：
   ```text
   http://您的电脑IP:8000/index.html
   ```

## 🛠️ 技术亮点
* **零依赖（Zero Dependencies）**：纯原生前端技术栈，不依赖 Pixi.js 或 Phaser 等任何游戏引擎，加载速度极快。
* **🔥 动态道具池下架机制（新）**：重构了商店抽取逻辑，引入唯一的 `id` 检索与 `splice` 内存裁剪技术。玩家选中特定唯一武器后，该道具将自适应在运行时被移出数组，彻底杜绝无效重复刷出的情况。
* **⏳ 状态机暂停拦截机制**：重构了游戏主循环，引入 `isPaused` 状态。在打开商店时安全拦截所有物理、渲染与手势碰撞逻辑，保障了动态加载 UI 时的流畅切换。
* **斜向速度归一化**：在 PC 端同时按下两个方向键时，位移速度经过数学公式归一化处理，避免了传统小游戏斜向移动速度变快的 BUG。
* **多端自适应布局**：CSS 采用百分比与视口单位（vw/vh）控制大小，并在移动端通过 `touch-action: none` 彻底禁用了页面滑动与手势缩放，带来纯正的独立 App 游玩体感。
