# 🥔 AI 土豆兄弟 (Brotato Lite)

一个基于 HTML5 Canvas 和纯原生 JavaScript 开发的轻量级肉鸽（Roguelike）割草小游戏。项目采用全解耦的分体架构设计，无任何第三方依赖，完美适配 PC 端（键盘）与移动端（手机虚拟摇杆）双端操作。

## 🎮 游戏核心玩法
1. **走位生存**：红色敌人会从四周源源不断地涌向中央，玩家需要控制黄色土豆进行走位躲避。
2. **自动锁敌**：角色会自动锁定并射击距离最近的敌人，无需手动瞄准。
3. **磁性收集**：消灭敌人后会掉落绿色经验石，当玩家靠近时，经验石会自动产生磁性并向玩家飞行。
4. **升级成长**：收集足够的经验石即可升级。每升一级，角色的射速都会大幅度加快，并获得一定的生命值回复。

## 🚀 如何运行游戏？

### 💻 途径一：电脑端直接运行（最简单）
直接双击 `index.html` 文件，它会自动通过您电脑的默认浏览器（如 Edge, Chrome）打开，使用键盘 **`W, A, S, D`** 或 **方向键** 即可控制移动。

### 📲 途径二：手机端局域网畅玩（推荐测试触控）
如果您安装了 Python 环境，可以通过以下步骤将电脑变为局域网服务器，用手机连接游玩：

1. 进入 `brotato_game` 文件夹，在地址栏输入 `cmd` 并回车打开命令行窗口。
2. 在窗口中输入以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
   *(注：若提示错误，可尝试使用快捷命令 `py -m http.server 8000`)*
3. 保持该窗口不要关闭，重新打开一个全新的 `cmd` 窗口，输入 `ipconfig` 查询您电脑的 **IPv4 地址**（例如 `192.168.1.5`）。
4. 确保手机与电脑连接在**同一个 Wi-Fi** 下，在手机浏览器中输入网址即可开始游玩：
   ```text
   http://您的电脑IP:8000/index.html
   ```

## 🛠️ 技术亮点
* **零依赖（Zero Dependencies）**：纯原生前端技术栈，不依赖 Pixi.js 或 Phaser 等任何游戏引擎，加载速度极快。
* **斜向速度归一化**：在 PC 端同时按下两个方向键时，位移速度经过数学公式归一化处理，避免了传统小游戏斜向移动速度变快的 BUG。
* **多端自适应布局**：CSS 采用百分比与视口单位（vw/vh）控制大小，并在移动端通过 `touch-action: none` 彻底禁用了页面滑动、手势缩放及长按弹出菜单，带来纯正的独立 App 游玩体感。
